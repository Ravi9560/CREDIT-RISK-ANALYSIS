# CREDIT-RISK-ANALYSIS
Interactive Power BI dashboard to analyze loan default risk based on age, loan intent, home ownership, and loan grade.

**Project Overview**
This project focuses on analyzing borrower data to identify patterns in loan default risk using Power BI.

**Objective**
To understand how different factors like age, loan intent, home ownership, and loan grade impact default risk.

**Key Insights**
Younger borrowers (20–25) show relatively higher default risk
Mid-age group (26–35) has the lowest default tendency
Loan intent and home ownership significantly influence borrower distribution

**Dashboard Features**
Default Rate Analysis by Age Group
Loan Status Distribution by Loan Intent
Loan Status Distribution by Home Ownership
Loan Status Distribution by Loan Grade
KPI Metrics (Total Borrowers, Default Risk, Avg Loan Amount, Avg Interest Rate)
